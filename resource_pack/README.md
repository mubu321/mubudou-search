# flakeykey810407 Skin Replacer

統合版マインクラフト用のリソースパックです。

## 使い方

1. `resource_pack` フォルダをZIPファイルにまとめます
2. マインクラフト統合版でローカルワールドを開きます
3. ワールド設定 → リソースパック → 追加 → このファイルを選択
4. リソースパックを有効化してワールドを起動

## 機能

- flakeykey810407 さんのスキンがクリーパースキンに見えるようになります
- あなたのクライアント側のみの表示変更です
- 他のプレイヤーのスキンには影響しません

## ファイル構成

```
resource_pack/
├── pack_manifest.json      # リソースパック設定
├── pack_icon.png          # リソースパックのアイコン
└── textures/entity/player/
    ├── flakeykey810407.png # flakeykey810407さんのスキン
    ├── steve.png          # デフォルトスキン
    └── alex.png           # デフォルトスキン
```
